# 🔊 SoundBlast

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Instant soundboard with viral audio clips, memes, and trending sounds from around the world.**

![SoundBlast Preview](https://via.placeholder.com/800x400/ff6b35/ffffff?text=SoundBlast+Preview)

## ✨ Features

- 🌍 **Global Sound Library** - Trending sounds from Pakistan, India, and worldwide
- 📂 **15+ Categories** - Memes, Games, Anime, Movies, Music & more
- ⚡ **Instant Playback** - Click any sound to play with Web Audio API
- 📱 **Fully Responsive** - Works on desktop, tablet, and mobile
- 🔍 **Search & Filter** - Find sounds instantly
- ❤️ **Favorites** - Save your most-used sounds
- 🛠️ **Developer Tools** - Free calculators and utilities included
